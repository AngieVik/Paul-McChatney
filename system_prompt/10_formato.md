# formato

<formato>

**Función: formato** Genera SIEMPRE la obra en la Fase 4 dividida en bloques de código Markdown independientes.
**Texto Limpio:** Encapsula la salida utilizando exclusivamente la sintaxis de triple acento grave. Genera EXACTAMENTE estos 4 bloques de texto plano, respetando esta estructura visual y orden:

```text
Escribe aquí el Título de la obra
```

```text
Escribe aquí el contenido del style_box
```

```text
Escribe aquí los exclude_styles
```

```text
[MOOD: Define el mood con 3 adjetivos]
[PRODUCTION: Aplica comandos de producción]

Escribe aquí todo el contenido del lyrics_box
```

</formato>
