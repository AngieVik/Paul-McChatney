Prompts de estilo (Lyrics Box) Para guiar a la IA en la estructura, usa marcadores entre corchetes, por ejemplo:
[Intro: fast frantic piano solo]

### Control de Voz y Rango Vocal

Para definir con precisión el tipo de cantante y su registro:

* **Voces Masculinas Graves:**
  * **`Bajo de E2 a E4`**: Para un tono muy profundo.
  * **`Varitono de G2 a G4`**: Para una voz con cuerpo y fuerza, un poco más alta.
  * **`Tenor de C3 a C5`**: Para voces limpias, melódicas y más agudas.
* **Voces Femeninas (Estilo "No Diva" / Íntimo):**
  * **`Voz femenina registro muy bajo`**.
  * **`Susurrada`** / **`Íntima`**.
  * **`Tesitura solo rango bajo`**.
  * **`Límite de rango vocal E3 a A3`** / **`No exceder`**,.

### Dinámica y Volumen (Intensidad)

Etiquetas para controlar la fuerza de la interpretación:

* **`Dinámica P`** (Piano): Para momentos íntimos, contenidos, tipo susurro melódico y sin clímax.
* **`MF`** (Mezzoforte): Para una intensidad moderadamente fuerte.
* **`F`** (Forte): Para cantar fuerte y con proyección.

### Técnica y Estilo Vocal

Instrucciones específicas sobre cómo debe cantar la IA:

* **`Sotoboce`** (Sotto voce): Cantar en voz baja o susurrando.
* **`Sin vibrato`**: Para una voz más plana y directa.
* **`Sin ad libs`**: Para evitar improvisaciones vocales extras.
* **`Emoción desapegada`** / **`Internalizada`**: Para interpretaciones más frías o introspectivas.

### Efectos y Texturas (Instrumental y Mezcla)

Etiquetas para añadir procesado de audio y ambiente:

* **`Robotic Boys`** (Robotic Voice): Efecto para las voces.
* **`Crujido de vinilo`**: Para dar textura retro a la instrumental.
* **`Ambiente urbano`**: Capas de sonido para el entorno.
* **`Drop effect`**: Efecto de mezcla para marcar transiciones.

### Coros y Armonías

Prompts para simular capas de voces y producción profesional:

* **`Coro armonizado`**.
* **`Voces femeninas superpuestas`**.
* **`Voces de fondo suaves`**.

### Etiquetas de Exclusión (Exclude Styles)

Términos que se deben poner en el campo de "Exclude Styles" para evitar resultados no deseados:

* **Para asegurar voces graves masculinas:** Excluir `soprano`, `mezzo`, `alto`.
* **Para asegurar voces femeninas íntimas (no diva):** Excluir `Belting` (cantar con mucha potencia), `melismas` (adornos vocales rápidos), `soprano`, `alto`.
